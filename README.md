- [DEMO LINK](https://MakhmudKerimov.github.io/MyBike/)

## Description

The project is a landing page for a bike business, designed to attract customers and showcase the products. The layout includes sections with information about bicycles, their benefits, product details, and a feedback form. The landing page features a responsive design for user convenience across various devices.

## Technologies

- HTML5 — semantic page structure
- SCSS (Sass) — variables, mixins, nesting
- CSS Grid & Flexbox — responsive layout
- Figma — layout and design
- NPM — dependency management

## Features

- Responsive layout (mobile, tablet, desktop)
- Pixel-perfect implementation from Figma
- Smooth scrolling navigation
- Contact / feedback form

# Setup & Installation

Follow these steps to run the project locally:

1. Clone the repository
   git clone https://github.com/MakhmudKerimov/MyBike.git

2. Install dependencies
   npm install or yarn install

3. Run the project locally
   npm start or yarn start
